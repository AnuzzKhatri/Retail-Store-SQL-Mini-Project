# 🛒 Retail Store SQL Mini Project

## 📌 Project Overview

This project demonstrates SQL fundamentals and advanced querying techniques using a Retail Store Database. The project covers database creation, data manipulation, analytical queries, joins, subqueries, aggregations, and reporting.

The project was developed as part of a SQL Mini Project to strengthen practical database management and data analysis skills using MySQL.

---

## 🎯 Objectives

- Design and manage a relational database
- Perform CRUD operations
- Retrieve business insights using SQL
- Practice analytical SQL queries
- Understand relationships between multiple tables
- Improve database querying skills
